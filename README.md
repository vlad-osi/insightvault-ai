# InsightVault AI

AI-powered knowledge base with Retrieval-Augmented Generation (RAG).

## Overview

InsightVault AI allows teams to upload documents, search information using semantic search and communicate with their knowledge base using Large Language Models.

## Features

- Document upload
- AI-powered search
- RAG pipeline
- Source citations
- Retrieval trace
- Multi-provider LLM support
